View the project live here: [https://bigstepper69.github.io/portfolio/](https://bigstepper69.github.io/portfolio/)

## ✨ Key Features
* **Multipage Architecture**: 4 distinct pages (Home, About, Projects, Contacts) for clear navigation.
* **Responsive Design**: Built with **Bootstrap 5** to ensure perfect viewing on mobile, tablet, and desktop.
* **Interactive UI**: Includes a particle background and a custom JavaScript chatbot.
* **Engineering Tools**: Features a live network diagnostic tool and embedded video demonstrations of software projects.

## 🛠️ Tech Stack
* **Frontend**: HTML5, CSS3, JavaScript (ES6)
* **Framework**: Bootstrap 5
* **Hosting**: GitHub Pages
* **Tools**: VS Code, Git

## 📂 Repository Contents
* `index.html` - The landing page / Home.
* `portfolio.html` - Detailed project showcase.
* `style.css` - Custom styling and theme overrides.
* `*.js` - Logic for the chatbot and particle effects.
